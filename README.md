## `$ cat HOW_IT_WAS_BUILT.md`

This project was built with a modern AI-assisted workflow — from design to code.

### Design & Ideation — [Stitch by Google](https://stitch.withgoogle.com/)

> *"Design first. Code second."*

The entire visual identity — layout, color system, component hierarchy, and UI specs — was ideated and designed using **Stitch**, Google's AI design tool. Stitch generated high-fidelity screens from natural-language prompts, which were synthesized into `DESIGN.md` to drive every styling decision in the codebase.

**Stitch MCP** was connected directly to Claude Code using the [stitch-skills](https://github.com/google-labs-code/stitch-skills) package, enabling a tight loop between design generation and implementation:

```bash
# Install Stitch skills via Claude Code MCP
# → https://github.com/google-labs-code/stitch-skills
```

The `stitch_profile_the_code_canvas_terminal_minimalist_prd/` folder contains the original PRD generated by Stitch, and `DESIGN.md` is the design system synthesized from it.

### Coding — [Claude Code](https://claude.ai/code)

The implementation was built with **Claude Code** as a pair programmer — scaffolding components, wiring routing, generating TypeScript data types, and iterating on the terminal/IDE aesthetic end-to-end.

```
prompt → Stitch screens → DESIGN.md ──→ Claude Code → khaled-portfolio
              ↑                                ↑
         Stitch MCP                    stitch-skills MCP
```

## `$ cat DESIGN.md | head -5`

The entire UI is a **VSCode-as-identity** experience:
- `#090A0F` Void Black backgrounds
- `#39FF14` Terminal Green — the only color that commands attention
- `#00E5FF` Cyber Cyan — for links, JSON keys, and secondary accents
- `Fira Code` everywhere code lives. `Space Grotesk` where titles breathe.
- 0px border radius. Always. (Pill badges are the one exception — and they *earn* it.)
